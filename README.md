# WeatherApp
This is an weather web app which gives information about live weather.\
Deployed on heroku.\
Link for the site -: https://weatherspy.herokuapp.com

Frontend -:

> HTML,
> CSS,
> JAVASCRIPT

Backend -:

> Node.js,
> Express.js,
> Embedded JavaScript

Frameworks -:

> Bootstrap,
> Tailwind

Api used - https://openweathermap.org/api
